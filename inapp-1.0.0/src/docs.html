Documentación del sistema

Guía básica de uso del dashboard Sanos y Salvos, enfocado en reportar y encontrar mascotas perdidas.

Ir al dashboard

Descripción general

Sanos y Salvos es una maqueta de sistema orientada al registro, seguimiento y difusión de mascotas perdidas. Su objetivo es ayudar a los tutores a reportar una mascota extraviada y facilitar que otros usuarios puedan colaborar con avistamientos o información útil para encontrarla.

La plantilla original de ThemeWagon fue adaptada al contexto del proyecto, reemplazando los elementos de inventario y ventas por módulos relacionados con mascotas reportadas, casos activos, avistamientos, zonas de búsqueda y mascotas encontradas.

Tecnologías usadas

  • HTML
  • Bootstrap 5
  • CSS / SCSS
  • Vite
  • Plantilla ThemeWagon adaptada

Dashboard

Muestra indicadores principales como mascotas reportadas, casos activos, mascotas encontradas y reportes urgentes.

Mascotas

Permite revisar el listado de mascotas perdidas, encontradas, avistadas o en búsqueda.

Registrar mascota perdida

Contiene un formulario para ingresar datos de la mascota, tutor, comuna, fecha de pérdida, descripción y fotografía.

Reportes de búsqueda

Presenta resúmenes de búsqueda, avistamientos, casos urgentes, mascotas encontradas y actividad por comuna.

Uso básico del dashboard

  1. Ingresar al panel principal desde Dashboard.
  2. Revisar los indicadores generales de mascotas reportadas, encontradas y en búsqueda.
  3. Acceder a Mascotas para consultar el listado de reportes registrados.
  4. Usar Registrar mascota perdida para simular el ingreso de un nuevo caso.
  5. Revisar Reportes de búsqueda para analizar estados, zonas y actividad reciente.

Instalación y ejecución

Para ejecutar la plantilla modificada, se debe abrir la terminal dentro de la carpeta del proyecto: 

cd inapp-1.0.0

Luego se instalan las dependencias:

npm install

Finalmente se inicia el servidor local:

npm run dev

Cambios realizados sobre la plantilla

Archivo Antes Después
index.html Dashboard de inventario y ventas Dashboard de mascotas perdidas y reportes de búsqueda
inventory.html Inventario de productos Listado de mascotas reportadas
create-product.html Formulario para agregar productos Formulario para registrar una mascota perdida
reports.html Reportes de ventas e inventario Reportes de búsqueda, avistamientos y mascotas encontradas
signin.html / signup.html Acceso genérico de la plantilla Inicio de sesión y registro de tutores o voluntarios
Nota: Se eliminó el uso de ApexCharts, ya que no era una librería trabajada en clases. Los gráficos fueron reemplazados por cards, tablas y barras de progreso usando Bootstrap.
